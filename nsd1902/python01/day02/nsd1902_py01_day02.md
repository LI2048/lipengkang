## 判断语句

### 条件表达式，也叫三元运算符

```python
>>> a = 10
>>> b = 20
>>> if a <= b:
...     smaller = a
... else:
...     smaller = b
... 
>>> smaller
10
>>> s = a if a <= b else b
>>> s
10
```

random.choice方法：在序列对象中选择一项

```python
>>> import random
>>> random.choice('abcdef')
'e'
>>> random.choice('abcdef')
'c'
>>> random.choice(['aaa', 'bbb', 'ccc'])
'bbb'
>>> random.choice(['aaa', 'bbb', 'ccc'])
'aaa'
>>> random.choice(['aaa', 'bbb', 'ccc'])
'ccc'
```
