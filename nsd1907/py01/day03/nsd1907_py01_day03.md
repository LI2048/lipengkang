### 读取文本文件

```python
(nsd1907) [root@room8pc16 day03]# cp /etc/passwd /tmp/
# 默认以r方式打开，文件不存在则报错

>>> data = f.read()  # 读取全部内容，保存到data中
>>> print(data)
>>> data = f.read()  # 文件指针已到结尾，再读取将为空串

''
>>> f.close()   # 关闭文件
```
