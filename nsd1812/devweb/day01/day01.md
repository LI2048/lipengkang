## html基本格式

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>my html</title>
</head>
<body>

</body>
</html>
```

## 元素

也叫标签、标记。分为封闭型的双标记和单标记。

- 块级元素：不管数据量多少，至少占一整行，如标题元素h1-h6、p、div
- 行内元素：不会产生换行
